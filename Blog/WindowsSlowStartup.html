My computer boots slower and slower since last month. Today it took more than 20 minutes to start so I decided to solve this problem.

After trying nearly every solution I could find online, I found a super large temp file (more than 150 GB) in AppData/Local/Temp. First I delete its contents with shift+del, but failed without any error reported. I also tried cleanmgr for Windows but still no luck. So I wrote a simple python script and it works well (and faster):

``` import os import shutil def delFiles(path): for file in os.listdir(path): try: if os.path.isdir(path+"\\"+file): print(path+"\\"+file) shutil.rmtree(path+"\\"+file) except: # when access denied continue ```
